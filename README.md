# resume-navigator

I want to build an application that acts like an Applicant Tracking System. This app will have a storefront resume building page for candidates, and in the backend for the recruiter side I should be able to list and filter the candidates based on the properties captured in the resumes.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/resume-navigator.git
cd resume-navigator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
